# LumenWatch Agent

LumenWatch is a Python service for website change intelligence. The first
stage contains a FastAPI application skeleton and health endpoint.

## Quick start

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -e ".[dev]"
.\start.ps1
```

Open `http://127.0.0.1:8020/docs`.
